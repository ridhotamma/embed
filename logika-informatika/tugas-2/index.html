Soal 1 (30 poin)

Perhatikan kalimat berikut ini:

$$ F: if (P \land Q) \text{ then } ((P \land Q) \lor S) $$

Apakah kalimat tersebut merupakan kalimat yang bersifat valid? Buktikan dan jelaskan dengan menggunakan metode Proof by Falsification (PbF)!

Solusi:

Langkah 1: Identifikasi struktur kalimat

Bentuk: If A then B, dimana:

  • A = $(P \land Q)$
  • B = $((P \land Q) \lor S)$

Langkah 2: Asumsi kalimat tidak valid

Untuk PbF, kita asumsikan kalimat bernilai false, artinya:

  • A = true
  • B = false

Langkah 3: Analisis

  1. Jika A = true, maka $P \land Q = true$
    • P = true
    • Q = true
  2. Jika B = false, maka $((P \land Q) \lor S) = false$
    • $(P \land Q) = false$ dan $S = false$
  3. Terjadi kontradiksi karena:
    • Dari (1), $(P \land Q) = true$
    • Dari (2), $(P \land Q) = false$

Kesimpulan:

Karena terjadi kontradiksi saat diasumsikan kalimat bernilai false, maka kalimat tersebut terbukti valid.

Soal 2 (35 poin)

Buktikan validitas kalimat:

$$ if \text{ true then } ((\neg Q \land P) \lor (Q \lor \neg P)) $$

dengan menggunakan pohon semantik!

Solusi:

Langkah 1: Negasi kalimat

Untuk membuktikan dengan pohon semantik, kita negasikan kalimat:

$$ \neg(if \text{ true then } ((\neg Q \land P) \lor (Q \lor \neg P))) $$ $$ true \land \neg((\neg Q \land P) \lor (Q \lor \neg P)) $$

Soal 3 (35 poin)

Tentukan hasil substitusi berikut ini:

$$ \{(if \text{ }Q \text{ then } (R \land S)) \iff \neg(P \land \neg(R \lor S))\} \triangleleft $$ $$ \{\neg(R \land S) \leftarrow R, Q \leftarrow (P \lor Q), (P \land R) \leftarrow (Q \lor S)\} $$

Solusi:

Langkah 1: Identifikasi substitusi

  • $\neg(R \land S)$ diganti dengan $R$
  • $Q$ diganti dengan $(P \lor Q)$
  • $(P \land R)$ diganti dengan $(Q \lor S)$

Langkah 2: Lakukan substitusi

Substitusi secara berurutan:

$$ \{(if \text{ }(P \lor Q) \text{ then } (R \land S)) \iff \neg((Q \lor S) \land \neg(R \lor S))\} $$

Hasil Akhir:

$$ \{(if \text{ }(P \lor Q) \text{ then } (R \land S)) \iff \neg((Q \lor S) \land \neg(R \lor S))\} $$
