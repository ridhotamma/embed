Soal 1 (30 poin)

Diketahui dua kalimat logika proposisional:

F: if true then (not P and Q) else (Q or not S)
G: not (P or Q) and (if P then R else (P and not Q))
P Q R S F G
T T T T F F
T T F T F F
T F T T F F
F T T T T F
F F T T T T

Penjelasan:

  • Untuk F: Karena kondisi adalah "true", maka selalu mengevaluasi (not P and Q)
  • Untuk G: Ekspresi akan bernilai true hanya jika both P dan Q false, dan kondisional kedua true

Soal 2 (40 poin)

Membuktikan ekuivalensi:

F: if (P and Q) then R
G: if (not R) then (not P or not Q)
P Q R P and Q F not R not P or not Q G
T T T T T F F T
T T F T F T T F
T F T F T F T T
F T F F T T T T

Kesimpulan:

Dari tabel kebenaran di atas, dapat dilihat bahwa F dan G memiliki nilai kebenaran yang sama untuk setiap kombinasi input P, Q, dan R. Ini membuktikan bahwa kedua pernyataan tersebut ekuivalen secara logis.

Soal 3 (30 poin)

Analisis menggunakan pohon semantik:

  1. Jika R true: Kalimat menjadi true karena implikasi bernilai true ketika anteseden false
  2. Jika R false:
    • R and P akan selalu false
    • not P ↔ false akan bernilai true jika P true
    • Ini membuat seluruh implikasi bernilai true

Kesimpulan:

Kalimat F adalah sebuah tautologi karena selalu bernilai true untuk semua kemungkinan nilai R dan P.
