Solusi Soal Vektor

Soal 1: u + v - w

Diketahui:

  • u = (1,2,3)
  • v = (2,-3,1)
  • w = (0,-3,5)

Langkah penyelesaian:

  1. u + v = (1+2, 2+(-3), 3+1) = (3, -1, 4)
  2. (u + v) - w = (3, -1, 4) - (0, -3, 5)
  3. Hasil akhir = (3-0, -1-(-3), 4-5) = (3, 2, -1)

Jawaban: C. (3,2,-1)

Soal 2: Kombinasi Linear u = (0,1,0)

Diketahui: A = {(1,0,1), (0,2,0)}

Langkah penyelesaian:

  1. Misalkan u = a(1,0,1) + b(0,2,0)
  2. Maka: (0,1,0) = (a, 2b, a)
  3. Dari persamaan tersebut:
    • a = 0 (dari x)
    • 2b = 1 (dari y)
    • a = 0 (dari z)
  4. Diperoleh b = 1/2
  5. Karena ada solusi (a=0, b=1/2), maka u merupakan kombinasi linear dari A

Jawaban: Ya, vektor u merupakan kombinasi linear dari A

Soal 3: Kombinasi Linear v = (1,-1,1)

Diketahui: A = {(1,0,1), (0,2,0)}

Langkah penyelesaian:

  1. Misalkan v = a(1,0,1) + b(0,2,0)
  2. Maka: (1,-1,1) = (a, 2b, a)
  3. Dari persamaan tersebut:
    • a = 1 (dari x)
    • 2b = -1 (dari y)
    • a = 1 (dari z)
  4. Diperoleh b = -1/2
  5. Nilai a dan b konsisten, maka v merupakan kombinasi linear dari A

Jawaban: Ya, vektor v merupakan kombinasi linear dari A

Soal 4: Himpunan K membangun R³

Diketahui: K = {(1,1,1), (2,2,0), (3,0,0)}

Langkah penyelesaian:

  1. Untuk membangun R³, himpunan vektor harus:
    • Memiliki 3 vektor (syarat perlu)
    • Ketiga vektor harus bebas linear
  2. Mari cek determinan matriks yang dibentuk dari vektor-vektor tersebut:
    |1 1 1| |2 2 0| = (1)(2)(0) + (1)(0)(3) + (1)(2)(0) - (1)(2)(3) - (1)(2)(0) - (1)(0)(0) = -2 |3 0 0|
  3. Karena determinan ≠ 0, maka ketiga vektor bebas linear dan dapat membangun R³

Jawaban: A. Benar

Soal 5: Himpunan L membangun R³

Diketahui: L = {(2,-1,3), (4,1,2), (8,-1,8)}

Langkah penyelesaian:

  1. Untuk membangun R³, kita perlu cek apakah ketiga vektor bebas linear
  2. Mari cek determinan matriks:
    |2 -1 3| |4 1 2| = (2)(1)(8) + (-1)(2)(8) + (3)(4)(-1) - (3)(1)(8) - (2)(2)(-1) - (-1)(4)(8) |8 -1 8|
  3. Setelah dihitung, determinan = 0
  4. Karena determinan = 0, maka ketiga vektor tidak bebas linear
  5. Oleh karena itu, himpunan L tidak dapat membangun R³

Jawaban: B. Salah

Soal 6: Himpunan U = {x | x = (a,1,0), a ∈ R}

Langkah penyelesaian:

  1. Himpunan U berisi semua vektor dengan bentuk (a,1,0) dimana a adalah bilangan real
  2. Ini membentuk garis dalam R³ yang:
    • Komponen y selalu 1
    • Komponen z selalu 0
    • Komponen x bisa berupa bilangan real apapun
  3. Karena hanya bisa bergerak di satu arah (sepanjang sumbu x), ini merupakan ruang bagian dari R³

Jawaban: A. Benar

Soal 7: Himpunan V = {x | x = (a,2a,-a), a ∈ R}

Langkah penyelesaian:

  1. Himpunan V berisi vektor-vektor dengan bentuk (a,2a,-a)
  2. Ini dapat ditulis sebagai a(1,2,-1) untuk semua a ∈ R
  3. Ini membentuk garis lurus yang melalui origin dengan arah vektor (1,2,-1)
  4. Karena hanya bisa bergerak sepanjang satu arah, ini merupakan ruang bagian dari R³

Jawaban: A. Benar

Soal 8: Himpunan V = {x = (x,y,z), x - 2z = 0; x,y,z ∈ R}

Langkah penyelesaian:

  1. Persamaan x - 2z = 0 dapat ditulis sebagai x = 2z
  2. Maka setiap vektor dalam V berbentuk (2z,y,z) dimana y,z ∈ R
  3. Ini membentuk bidang dalam R³ karena:
    • y dapat bergerak bebas
    • z dapat bergerak bebas
    • x tergantung pada z
  4. Karena membentuk bidang, ini merupakan ruang bagian dari R³

Jawaban: A. Benar

Soal 9: Himpunan W = {x = (x,y,z), x - yz = 0; x,y,z ∈ R}

Langkah penyelesaian:

  1. Persamaan x - yz = 0 adalah persamaan non-linear (karena ada perkalian yz)
  2. Untuk menjadi ruang bagian, himpunan harus memenuhi:
    • Tertutup terhadap penjumlahan
    • Tertutup terhadap perkalian skalar
  3. Ambil v₁ = (y₁z₁,y₁,z₁) dan v₂ = (y₂z₂,y₂,z₂)
  4. v₁ + v₂ = (y₁z₁+y₂z₂,y₁+y₂,z₁+z₂)
  5. Hasil penjumlahan tidak memenuhi x = yz
  6. Karena tidak tertutup terhadap penjumlahan, W bukan ruang bagian

Jawaban: B. Salah

Soal 10: Himpunan vektor P bebas linear

Langkah penyelesaian:

  1. Himpunan P terdiri dari vektor-vektor: v₁ = (1,0,-1), v₂ = (1,1,0), v₃ = (1,3,1)
  2. Untuk mengecek kebebasan linear, kita hitung determinan:
    |1 1 1| |0 1 3| = 1(1)(1) + 1(3)(-1) + 1(0)(1) - 1(1)(1) - 1(3)(1) - 1(0)(-1) = 1 - 3 + 0 - 1 - 3 + 0 = -6 |-1 0 1|
  3. Karena determinan ≠ 0, ketiga vektor bebas linear

Jawaban: A. Benar

Soal 11: Himpunan vektor Q bebas linear

Langkah penyelesaian:

  1. Himpunan Q terdiri dari vektor-vektor: v₁ = (1,0,-2), v₂ = (1,1,-2), v₃ = (-2,3,4)
  2. Hitung determinan:
    |1 1 -2| |0 1 3| = 1(1)(4) + 1(3)(-2) + (-2)(0)(1) - (-2)(1)(3) - 1(3)(1) - 1(0)(-2) = 4 - 6 + 0 + 6 - 3 + 0 = 1 |-2 -2 4|
  3. Karena determinan ≠ 0, ketiga vektor bebas linear

Jawaban: A. Benar

Soal 12: B = {u,v,w} ⊂ R³ bebas linear, buktikan {u-v+w, -u+w} ⊂ R³ juga bebas linear

Langkah penyelesaian:

  1. Misalkan a(u-v+w) + b(-u+w) = 0
  2. Kelompokkan koefisien:
    • u: (a-b)
    • v: (-a)
    • w: (a+b)
  3. (a-b)u + (-a)v + (a+b)w = 0
  4. Karena {u,v,w} bebas linear, maka:
    • a-b = 0
    • -a = 0
    • a+b = 0
  5. Dari sistem persamaan ini, diperoleh a = 0 dan b = 0
  6. Karena hanya ada solusi trivial, {u-v+w, -u+w} bebas linear

Soal 13: Tunjukkan bahwa P basis di R³

Langkah penyelesaian:

  1. Untuk menjadi basis di R³, himpunan P harus:
    • Memiliki 3 vektor
    • Vektor-vektor tersebut bebas linear
    • Merentang R³
  2. P memiliki 3 vektor: v₁ = (1,0,-1), v₂ = (1,1,0), v₃ = (1,3,1)
  3. Hitung determinan untuk kebebasan linear:
    |1 1 1| |0 1 3| = -6 ≠ 0 |-1 0 1|
  4. Karena det ≠ 0:
    • Vektor-vektor bebas linear
    • Merentang R³
  5. Karena memenuhi semua syarat, P adalah basis untuk R³
